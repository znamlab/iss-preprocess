Crunch the data¶

The first step is to get the data from the microscope into usable form. This is done by iss project-and-average. This script will project the data into a single image and average the data across rounds and channels to generate illumination correction images. It will also generate ROI overviews that can be used to select the reference tiles.

Overview¶

The command is run as follows:

iss project-and-average --path relative/path/to/data

By default, the script will process only new files. There is a --force-redo flag that will force the script to re-run all steps, even if the output files already exist.
